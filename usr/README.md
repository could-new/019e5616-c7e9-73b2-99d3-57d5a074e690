# تطبيق إدارة المشاريع البرمجية

تطبيق حديث لإدارة ومتابعة المشاريع البرمجية، يقدم لوحة تحكم متجاوبة تعمل بشكل مثالي على أجهزة الجوال والويب وأجهزة سطح المكتب.

## الميزات الرئيسية
- **لوحة تحكم تفاعلية**: إحصائيات سريعة عن عدد المشاريع، المهام المنجزة، وساعات العمل.
- **عرض قائمة المشاريع**: متابعة حالة كل مشروع مع شريط التقدم لكل مشروع.
- **واجهة متجاوبة**: يتغير التصميم تلقائياً ليناسب حجم الشاشة، سواء كنت تستخدم هاتفاً محمولاً أو جهاز كمبيوتر.
- **تصميم عصري**: يعتمد على Material Design 3 للحصول على مظهر حديث ومريح.

## التقنيات المستخدمة
- Flutter Framework (للتطوير متعدد المنصات)
- Dart (لغة البرمجة الأساسية)
- Material Design 3

## كيفية التشغيل
للبدء في تشغيل التطبيق محلياً:
1. قم بتحميل وتثبيت Flutter SDK إذا لم يكن مثبتاً لديك.
2. قم بتنفيذ الأمر التالي في مجلد المشروع لجلب الحزم المطلوبة:
   ```bash
   flutter pub get
   ```
3. لتشغيل التطبيق على منصة معينة:
   ```bash
   flutter run
   ```
